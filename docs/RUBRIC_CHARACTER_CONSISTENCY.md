# Character Consistency Rubric

## Purpose
The Character Consistency rubric ensures that the main characters, companions, and recurring figures maintain a stable appearance throughout the entire book. This includes visual features, clothing, and their narrative roles.

## Scoring Criteria (1-5)
The Character Consistency score is assigned based on the following levels:

| Score | Rating | Description |
|---|---|---|
| 5 | Perfect Consistency | The character looks identical across all pages, including facial features, hair, and outfit. |
| 4 | High Consistency | Minor variations exist (e.g., slight change in hair volume) but the character is easily recognizable. |
| 3 | Moderate Consistency | Noticeable jitters in features or clothing, but still clearly the same character. |
| 2 | Poor Consistency | Character changes significant features (e.g., eye color, age) or outfit completely between pages. |
| 1 | Failure | Multiple characters appear where there should be one, or the character changes species or basic identity. |

## Evaluation Axes
Reviewers should assess consistency across these seven granular axes:

1.  **Visual Bible Reflection**: Does the character match the initial description/reference (`visualBible`)?
2.  **Character ID Consistency**: Is the same character consistently identified and tracked correctly?
3.  **Appearing Character Consistency**: Do the characters in the image match those listed for that page (`appearingCharacterIds`)?
4.  **Focus Character Consistency**: Is the main character of the page (`focusCharacterId`) visually prominent and correctly depicted?
5.  **Page-Level Linkage**: Does the image correctly reflect the interactions between characters described in the text?
6.  **Outfit & Hairstyle Consistency**: Do the clothes and hair remain stable across scenes?
7.  **Color Palette Consistency**: Does the character maintain their signature colors (e.g., "blue shirt," "yellow hat")?

**See also**: [Unintended Character Rubric](./QUALITY_RUBRICS/UNINTENDED_CHARACTER_RUBRIC.md) for detecting extra characters or duplication.

## Examples

### Good (Score 4-5)
- **Signature Items**: A character always wears the same red scarf.
- **Stable Features**: The child's face and hairstyle remain constant whether they are laughing or crying.
- **Companion Stability**: A companion dog always has the same spots and fur color.

### Poor (Score 1-2)
- **Identity Shift**: A character starts as a boy and suddenly appears as a girl on page 3.
- **Duplication**: Two versions of the main character appear in the same frame.
- **Wardrobe Chaos**: The character's clothes change color and style every single page without narrative reason.

## Nuances
- **Emotion & Angle**: Characters' faces will naturally change slightly when expressing strong emotions or when viewed from different angles (e.g., profile vs. front).
- **Growth/Change**: If the story explicitly mentions a change (e.g., "getting wet" or "putting on a coat"), these variations are expected and do not penalize the score.

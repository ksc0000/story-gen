# Investigate and Address `hiddenDetail` Becoming Primary Story Objective

## Context

The product roadmap for Phase 2: Story & Illustration Quality explicitly lists "hiddenDetail の主目的化防止" (Prevent `hiddenDetail` from becoming primary objective) as a key area for improvement. `hiddenDetail` is intended to add subtle, background elements to a story, enriching the narrative without overshadowing the main `storyGoal`. If `hiddenDetail` becomes a dominant plot point, it can detract from the story's coherence and quality. This task focuses on diagnosing instances where `hiddenDetail` over-influences the narrative and implementing targeted refinements.

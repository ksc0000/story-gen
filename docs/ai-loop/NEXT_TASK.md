## Context

The product roadmap for Phase 3: Template Mode, under "Quality Follow-ups," explicitly lists `IMG-001` as a "minor follow-up継続" task. This task aims to further enhance the image generation prompts for existing fixed templates by introducing more explicit negative prompt elements.

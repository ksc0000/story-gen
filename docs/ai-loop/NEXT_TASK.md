# Define Quality Rubric and Audit Process for `storyGoal` Adherence

## Context

The product roadmap for EhonAI emphasizes "Phase 2: Story & Illustration Quality" with a specific goal to ensure generated results are "acceptable picture books for sale." Within this phase, the roadmap explicitly lists "`storyGoal` の維持検証" (verify `storyGoal` adherence) as a currently unaddressed task under "本文品質" (story text quality). This task aims to formalize what `storyGoal` adherence means and how it can be consistently evaluated.
